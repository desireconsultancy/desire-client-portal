# Desire Consultancy - Client Portal

A premium B2B client portal enabling consultancy clients to track projects, manage documents, view certificates, monitor payments, and communicate with support efficiently.

## Features

- **Dashboard**: High-level overview of active projects, pending tasks, recent payments, and notification badges.
- **Project Tracking**: Comprehensive view of consultancy project steps, stages (e.g. FSSAI, BIS, Trademark), and completion status.
- **Document Vault**: Centralized and secure location for uploading and managing compliance documents.
- **Certificate Center**: Easy access to issued certificates and regulatory compliance documents.
- **Machine Orders & Payments**: Track client hardware orders and pay pending service/hardware invoices.
- **Support & Settings**: Direct contact line to the consultancy support agents and account personalization options.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or pnpm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```

3. Build the application for production:
   ```bash
   npm run build
   ```
